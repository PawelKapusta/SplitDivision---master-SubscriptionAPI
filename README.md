# SplitDivision---master-SubscriptionAPI

### Running the development server.

### To install

```bash
    npm install
```

### And run:

```bash
    npm run start
```

### To run project

#### Build docker image

```bash
    docker build -t master-subscription-api .
```

#### Run a container on port 5005

```bash
    docker run -p 5005:5005 master-subscription-api
```
